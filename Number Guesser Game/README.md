# بازی حدس عدد (Number Guesser)

یک بازی ساده‌ی خط فرمان که کاربر باید یک عدد تصادفی بین ۱ تا ۱۰۰ را حدس بزند.  
به ازای هر حدس اشتباه، یک امتیاز از کاربر کم می‌شود و یک راهنما (بالاتر/پایین‌تر) دریافت می‌کند.

---

## امکانات

- تولید عدد تصادفی بین ۱ تا ۱۰۰
- دریافت حدس کاربر با اعتبارسنجی ورودی (`try/except`)
- نمایش راهنمای «عدد بزرگ‌تر است» یا «عدد کوچک‌تر است»
- سیستم امتیازدهی (شروع از ۱۰۰ و کاهش به ازای هر اشتباه)
- امکان خروج از بازی با وارد کردن حرف `q`
- امکان بازی مجدد بعد از حدس درست

---

## پیش‌نیازها

- Python 3.10 یا بالاتر
